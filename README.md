# مراقبه رمضان

ابزار ترکینگ اعمال عبادی در ماه مبارک رمضان. کاربران هر روز می‌توانند کارهای روز را ببینند، تیک بزنند، از لینک‌های مستقیم به قرآن و مفاتیح استفاده کنند، و در تنظیمات زمان نوتیفیکیشن را انتخاب کنند.

**حریم خصوصی:** تمامی داده‌ها (پیشرفت روزانه، تنظیمات، نام) فقط روی همین دستگاه در مرورگر (localStorage) ذخیره می‌شوند و به هیچ سروری ارسال نمی‌شوند.

---

## پیش‌نیاز

- [Bun](https://bun.sh) (به‌جای Node/npm)

---

## اجرای محلی

```bash
cd app
bun install
bun run dev
```

اپ در حالت توسعه روی `http://localhost:3000` در دسترس است.

**بیلد و سرو نسخهٔ تولید:**

```bash
cd app
bun run build
bun run preview
```

یا برای سرو با اسکریپت سبک Bun:

```bash
bun run serve
```

---

## اجرا با داکر

از **روت پروژه**:

```bash
docker compose up --build
```

سپس اپ روی `http://localhost:3000` در دسترس است.

---

## ساختار پروژه

- **روت:** `docker-compose.yml`، `.gitignore`، `README.md`
- **`app/`:** تمام کد منبع و پیکربندی (React، Vite، TypeScript). بیلد داکر از همین پوشه انجام می‌شود.

---

## تکنولوژی

- React 19، TypeScript، Vite، Tailwind (CDN)، Heroicons
- بدون بک‌اند؛ بدون نیاز به API key یا فایل `.env` در نسخهٔ فعلی

---

## انتشار

- در محیط واقعی حتماً از **HTTPS** استفاده کنید (مثلاً از طریق reverse proxy یا پلتفرم deploy).
- فایل‌های `.env` و `.env.local` را در رپو قرار ندهید (در `.gitignore` هستند).
